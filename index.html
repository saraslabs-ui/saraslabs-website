Installation Guide

Install SARAS ERP Legacy step by step

Legacy edition requires MySQL 8.0. Database name must be exactly medical_erp.

01

Download ERP Setup

Download SARAS ERP Legacy setup from this website.

02

Install ERP

Run setup file and complete installation.

03

Download MySQL 8.0

Install MySQL Server 8.0 from Oracle official website.

Download MySQL
04

Set Password

During MySQL installation set root password as:

root
05

Create Database

Open MySQL Workbench and run:

CREATE DATABASE medical_erp;
06

Start ERP

Open SARAS ERP Legacy and connect to database:

medical_erp
