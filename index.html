Installation guide

How to install SARAS ERP Legacy.

Follow these steps carefully. Legacy edition requires MySQL 8.0 and database name must be medical_erp.

01

Download SARAS ERP Legacy

Download the Windows setup file from the official SARASLabs download button.

Download Setup
02

Install SARAS ERP

Run the downloaded setup file and complete the installation wizard.

03

Download MySQL Server 8.0

Install MySQL Server 8.0 from Oracle's official MySQL download page.

Download MySQL 8.0
04

Set MySQL root password

During MySQL installation, set the root password as:

root
05

Create database

Open MySQL Workbench or MySQL command line and create this database:

CREATE DATABASE medical_erp;
06

Start SARAS ERP Legacy

Open SARAS ERP Legacy and connect using MySQL database name:

medical_erp

Important database details

Database name must be exactly medical_erp. MySQL root password must be remembered because ERP database connection depends on it.
